# flutter_plus

框架主要包括：

1.hooks_riverpod：状态管理，provider的升级版本，详情自行查看官方文档，上面有介绍对比当前几种主流状态管理框架

2.retrofit：网络请求，熟悉Android的应该知道

3.freezed：实体类对象

还有一些用到的组件自行参考yaml文件


项目目前实现功能：

1.登录

2.复杂的布局动画（京东首页顶部，个人界面的动画，商品详情页的动画）

3.网络请求的下拉刷新和basepage管理页面状态

4.封装了很多复杂的部件，可以直接拿来使用
